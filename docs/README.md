# Documentation for Policy-Based Authorization System

This documentation is built with [VitePress](https://vitepress.dev/). 

## Local Development

```bash
cd docs
npm install
npm run docs:dev
```

## Build

```bash
npm run docs:build
```

The built documentation will be in `.vitepress/dist/`.

## Deployment

Documentation is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.

View live documentation: https://gsomenzi.github.io/policy-based-authorization-system/
